1、加最高权限

chmod 775  文件名

2、加可执行权限

chmod +x   文件名

3.进程
	查看进程：
	ps -ef|grep java
	杀死进程：
	kill -9 进程号

4.管道"|"
将一个命令的输出用作另一个命令的输入：
如：ls --help|more 分页查询帮助信息
	ps --ef|grep java 
5.&& 命令执行与控制
  命令之间使用&&连接，实现逻辑与的功能。
  只有在&&左边的命令返回真，右边的命令才会被执行
6.网络通讯命令
	ifconfig 显示或设置网络设备。
		ifconfig 显示网络设备
		ifconfig eth0 up 启用eth0网卡
		ifconfig eth0 down 停用eth0网卡
	ping
	netstat 查看网络端口。
		netstat -an|grep 3306 查询3306端口占用情况
